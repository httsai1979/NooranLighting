NOORAN LIGHTING - ASSET MANAGEMENT GUIDE

This project is set up to load assets dynamically based on the PRODUCT CODE.

1. DIRECTORY STRUCTURE
   Please place your files in the following folders inside the "public" directory:

   - Images:      public/assets/images/products/
   - Videos:      public/assets/videos/
   - Datasheets:  public/assets/datasheets/
   - Company Logo: public/assets/logos/

2. NAMING CONVENTION (CRITICAL)
   The system automatically looks for files matching the Product Code.
   
   Example Product:
   - Code: "MGT-REC-48V-2M"
   
   You should name your files as:
   - Image:      "MGT-REC-48V-2M.jpg"  (inside public/assets/images/products/)
   - Datasheet:  "MGT-REC-48V-2M.pdf"  (inside public/assets/datasheets/)
   - Video:      "MGT-REC-48V-2M.mp4"  (inside public/assets/videos/)

3. BULK UPLOAD
   You can drag and drop all your product images into the `public/assets/images/products/` folder at once.
   Ensure they possess the correct file extensions (.jpg for images, .pdf for datasheets).

4. UPDATING PRODUCT DATA
   To add more products, edit the file:
   src/data/products.js

   Copy an existing product block and update the 'id', 'code', 'title', and 'specs'.
